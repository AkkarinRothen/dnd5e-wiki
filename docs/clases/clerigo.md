## Rasgos Clave

### Preparar Conjuros
El clérigo puede preparar conjuros cada descanso largo. Conjuros preparados = **nivel de clérigo + mod. SAB**.
Los conjuros de dominio siempre están preparados (no cuentan para el límite).

### Canalizar Divinidad (nivel 2)
**2 usos** que se recuperan en **descanso corto**. Cada subclase define qué puede hacer.

**Opción universal disponible:**
- **Expulsar muertos vivientes**: Cada muerto viviente o infernal a ≤9 m que vea el símbolo sagrado debe hacer salvación de Sabiduría (CD de conjuro) o queda Asustado y alejándose de ti durante 1 minuto.

### Bendición Divina (nivel 7)
Cuando lanzas un conjuro de clérigo de nivel 1 o superior, puedes gastar espacios de conjuro adicionales para añadir 2d6 de daño radiante por nivel de espacio extra.

!!! note "El nombre exacto de este rasgo varía según la edición/traducción. Consulta tu nota de clase completa."
    
### Intervencion Divina (nivel 10)
Llamas a tu deidad para que intervenga. Efecto determinado por el DM según la situación.
